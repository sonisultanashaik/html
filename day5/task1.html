VISUAL STUDIO CODE

Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux.

Features of vs code:

1.Open source
2. Platform support
3. Development capabilities
4.Integrated Terminals
5. Extensions
6. Debugging

BASIC COMMANDS:

COMMANDS DESCRIPTIONS
Ctrl+S Saving the file
Ctrl+C Copying the text
ctrl+V Paste the text
Ctrl+X Cut the text
Ctrl+A Select all
Ctrl+Z For Redo
Ctrl+y For Undo
Contro + Increasing the font Or Zoom Out
Ctrl - Decreasing The Font Or Zoom In
Shift Up Moves Upwords
Shift down Moves Down words
Shift Left Moves Left words
Shift Right Moves Right Words
Alt+up It moves one level up
Alt+down It moves one level down
Shift+Alt+Down For copying the same line
Ctrl+Shift+Alt+Down For writing the code in multiple lines
Ctrl+Shift+N For new vs code window
Ctrl+Shift+P Ctrl+Shift+W
Ctrl+/ For comments
