EMMETS

Emmet is a set of plug-ins for text editors that allows for high-speed coding and editing in HTML, XML, XSLT, and other structured code formats via content assist. The project was started by Vadim Makeev in 2008 and continues to be actively developed by Sergey Chikuyonok and Emmet users. Wikipedia

EMMETS DESCRIPTION
! (or) html:5 For html5 structure
By tag name It is used to locate element with the given tag name
> Child For adding the child tag
+ sibling For adding the sibling
* multiple It is used to write similar tags more than one
{} It is used to write content
^ It is used to add the element to the parent
attribute="value" It is used to writ the attributes
